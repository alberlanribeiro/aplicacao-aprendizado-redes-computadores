Endereçamento IP

NAT

Network Address Translation - Tradução de Endereço de Rede é a técnica utilizada para traduzir um endereço IP privado em um endereço IP público.
Os endereços de ip privados podem ser utilizados para a criação de uma uma rede interna, evitando que cada equipamento nessa rede precise de um endereço público. Sendo assim, os endereços IP privados podem ser repetidos em várias redes privadas, só não podem ser roteados diretamente na internet. Para que a comunicação entre uma rede privada e a internet ocorra, é utilizado o NAT.

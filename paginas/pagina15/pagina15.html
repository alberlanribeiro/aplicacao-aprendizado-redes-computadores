Endereçamento IP

Classes de endereços IP (IPv4)

No IPv4, os endereços são divididos em classes que determinam a parte do endereço IP que identifica a rede e a parte que identifica os host, com isso sabemos a quantidade de hots possíveis em uma rede. O que diferencia uma classe da outra é o intervalo do primeiro octeto do endereço IP.

  • Classe A: Usada para redes muito grandes, com o primeiro octeto identificando a rede e os três restantes para dispositivos.

    Intervalo: 1.0.0.0 a 127.255.255.255 - 224 = 16.777.216 hosts.

  • Classe B: Adequada para redes de tamanho médio, com os dois primeiros octetos identificando a rede.

    Intervalo: 128.0.0.0 a 191.255.255.255 - 216 = 65.536 hosts.

  • Classe C: Usada em redes menores, com os três primeiros octetos identificando a rede.

    Intervalo: 192.0.0.0 a 223.255.255.255 - 28 = 256 hosts.

  • Classe D: Reservada para multicast.

    Multicast: é uma técnica em redes de computadores que, a partir de um único ponto, um conjunto de pacotes é enviado de forma simultânea para um grupo específico de dispositivos interessados.

    Intervalo: 224.0.0.0 a 239.255.255.255.

  • Classe E: Reservada para fins experimentais e não é amplamente usada.

    Intervalo: 240.0.0.0 a 255.255.255.255.

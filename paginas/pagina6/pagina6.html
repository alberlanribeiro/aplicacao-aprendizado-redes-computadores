Dispositivos de rede (continuação)

Ponto de acesso (access point)

O ponto de acesso é um dispositivo voltado para a distribuição wi-fi.
Ele não possui portas para distribuição via cabo.

Roteador

O roteador é um dispositivo que é capaz de conectar diferentes redes.
Ele pode receber internet de diferentes modos e distribuir via cabo ou wi-fi, sendo assim, o roteador possui um ponto de acesso embutido.
O que nos permite usar a internet é o roteador, que é capaz de lidar com os endereços de IP da internet e realizar a distribuição de endereços IP para uma rede interna.

Modem

Dispositivo que conecta o roteador com a rede da operadora de internet. Ele funciona modulando o sinal que chega da operadora de internet, podendo ser recebido pela linha de telefonia, ou por rádio, ou por fibra óptica e depois envia para o roteador.
Alguns dispositivos são capazes de fazer as duas funções juntando modem com ponto de acesso (wi-fi).

Endereçamento IP

O que é o endereço IP

Ao falar de endereçamento IP, podemos comparar o endereço IP com um endereço postal. Nesse sentido, o endereço IP funciona como um endereço eletrônico que permite que as informações sejam transmitidas pela internet, saindo da origem e chegando ao destino de maneira correta.

Versões e formatos do endereço IP

IPv4

É a versão mais antiga e amplamente utilizada. São representados como uma sequência de números decimais separados por pontos, por exemplo, "192.168.0.1". Um endereço IPv4 é composto por 32 bits ou 4 bytes, que são divididos em grupos chamados de octetos. Cada octeto equivale a 1 byte, que é representado por um número decimal que varia de 0 a 255. Esta versão suporta cerca de 4,3 bilhões de endereços, atualmente escassos devido ao crescimento das redes globais.

Byte: o byte é uma unidade de medida de armazenamento que representa 8 bits.
1 byte = 8 bits.

IPv6

Essa nova versão foi criada para resolver o problema de escassez dos endereços IPv4. São representados por 8 grupos de 4 dígitos hexadecimais separados por dois pontos, por exemplo, “FFED:0:0:0:0:BA98:3210:4562”. Cada grupo de 4 dígitos hexadecimais representa 16 bits de informações. Um endereço IPv6 pode ser descrito na forma encurtada, por exemplo, FFED:0:0:0:0:BA98:3210:4562, pode ser reescrito como, FFED::BA98:3210:4562, note que os zeros consecutivos são substituídos por "::".

Para fins de aprendizado vamos focar, por hora, no endereçamento IP na versão 4.

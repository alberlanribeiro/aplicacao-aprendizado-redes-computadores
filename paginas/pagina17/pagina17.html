Endereçamento IP

Alternativas para o problema de escassez

Alternativas foram tomadas para amenizar o problema, algumas delas são:

  • Definir faixas de endereço públicos e privados;
  • Criar a técnica de NAT (Network Address Translation - Tradução de Endereço de Rede);
  • Dividir as redes na notação CIDR (Classless Inter-Domain Routing - Encaminhamento inter-domínio sem classe).

Claro que tudo isso são apenas medidas temporárias, o problema só será totalmente resolvido com a total implementação do IPv6.

Endereços IP públicos e privados

Endereços IP privados

São usados em redes locais (LANs) e não são roteados diretamente na Internet. São reutilizáveis e destinados a identificar dispositivos dentro de uma rede privada.
Faixas de IP privados:
do 10.0.0.0 até 10.255.255.255
do 172.16.0.0 até 172.31.255.255
do 192.168.0.0 até 192.168.255.255.

Endereços IP públicos

São usados para identificar dispositivos na Internet global. Cada dispositivo conectado diretamente à Internet deve ter um endereço IP público único.
Os endereços IP públicos são todos aqueles que não estão nas faixas dos IPs privados, um exemplo é o famoso 8.8.8.8.

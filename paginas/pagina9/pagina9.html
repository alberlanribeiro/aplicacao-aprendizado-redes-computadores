Como funciona a comunicação entre dois computadores?

Tudo o que utilizamos em um computador é transformado em bit, logicamente quando realizamos a transferência de dados entre computadores também é através de bits.
Mas o que é um bit?

Bit: é a menor unidade de informação.
Bit é a simplificação de “Binary digit”
O bit só pode assumir dois valores: 0 ou 1, verdadeiro ou falso…

Lembra que falamos sobre PROTOCOLOS? Bem, imagine que um brasileiro tente se comunicar com um chinês… Difícil de entenderem-se, né? Para não dizer impossível…
Em redes seria a mesma coisa, diferentes máquinas de diferentes arquiteturas não conseguiriam se entender.
Para isso foram criados os protocolos e as pilhas de protocolos, que já comentamos antes. Eles atuam para padronizar a comunicação entre dois dispositivos.

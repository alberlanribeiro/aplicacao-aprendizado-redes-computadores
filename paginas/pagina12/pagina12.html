Camadas do modelo OSI (continuação)

4 - Camada de Transporte

A camada de transporte é responsável por fornecer meios para transmitir dados entre as duas partes conectadas. E também, faz o controle de qualidade de serviço. Os principais protocolos desta camada são o TCP e UDP.

3 - Camada de Rede

Responsável pelo roteamento e por enviar os dados entre redes diferentes. A camada de rede também determina o melhor caminho ou rota para os dados percorrerem. É nessa camada que o endereço lógico (endereço IP) de cada equipamento é estabelecido. Os principais protocolos desta camada são o IP e ICMP.

2 - Camada de Enlace de Dados

A camada de enlace de dados é referente à transmissão dos quadros de dados, ela é responsável de que a entrega dos dados seja feita sem erros e na ordem correta, isso entre duas máquinas que já estão conectadas.

1 - Camada Física

A camada física é a camada mais baixa do modelo OSI, é responsável pela transmissão dos bits e o meio físico em si que será utilizado para a transmissão.

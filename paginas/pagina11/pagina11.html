Camadas do modelo OSI

7 - Camada de Aplicação

É a camada mais alta, é responsável pela integração entre o usuário e a rede. Nessa camada é o único lugar em que o usuário consegue ter algum tipo de interação com a rede. Alguns dos protocolos desta camada são HTTP, FTP, DNS e DHCP.

6 - Camada de Apresentação

Na camada de apresentação os dados são traduzidos em uma forma que o aplicativo possa aceitar, ou seja, ela lida com a sintaxe e com a semântica das informações, essa conversão dos dados é fundamental para o receptor compreender. Nessa camada também é tratada a criptografia.

5 - Camada de Sessão

A camada de sessão é responsável por estabelecer, gerenciar e encerrar sessões de interação entre sistemas que se comunicam. Esta camada permite que aplicações troquem dados de forma sincronizada.
